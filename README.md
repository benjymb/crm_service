## Getting Started

### Prerequisites

This app requires:  
* python and pip
* Linux or Mac

### Set-up and usage

1. Clone the repo
   ```sh
   git clone https://github.com/benjymb/crm_service.git
   ```
2. Create a new virtual environment called venv
   ```sh
   python3 -m venv venv 
   ```
4. Install dependencies
   ```sh
   . venv/bin/activate
   pip install -r requirements.txt
   ```
5. Create database 
   ```sh
   python manage.py migrate
   ```
6. Run app
   ```sh
   python manage.py runserver
   ```
